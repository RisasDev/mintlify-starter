# Cómo actualizar un servidor de Hytale

1. Inicia el servidor desde el panel.
2. Ingresa al link indicado en la terminal.

Ejemplo:
```
https://oauth.accounts.hytale.com/oauth2/device/verify?user_code=CODIGO
```

<Frame>
  <img src="/images/games/hytale/actualizar-servidor.png"/>
</Frame>

3. Ingresa con tu cuenta de Hytale y aprueba la actualización.

<Frame>
  <img src="/images/games/hytale/actualizar-servidor-2.png"/>
</Frame>

4. Listo, regresa a la terminal y se te descargará la ultima versión de Hytale.
