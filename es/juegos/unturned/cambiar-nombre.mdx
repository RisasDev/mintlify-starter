## Cambiar el nombre desde el panel

1. Ve al **Panel de tu servidor**.
2. Entra en la sección **Startup**.
3. Busca el campo:
```
NAME
```
4. Escribe el nuevo nombre de tu servidor.
```
Ejemplo:
Survival Chile RP | Vanilla
```

5. Guarda los cambios.
6. Reinicia el servidor.
